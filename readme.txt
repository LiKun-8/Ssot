截图置顶 v1.0.1

使用说明：
    1、按shift+z，松开后拖动鼠标即可截图；
    2、双击关闭当前截图；
    3、托盘菜单-清除，清除所有截屏图片，同时可释放内存；
    4、最后一次截屏结果保存于剪切板，右键粘贴即可保存。
