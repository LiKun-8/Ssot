Ssot
    ——一款简单的截图工具

使用说明：
    1、按shift+z，松开后拖动鼠标即可截图；
    2、双击关闭当前截图；
    3、右击托盘菜单，点击“清除截图”，可清除所有截屏图片，同时可释放内存；
    4、最后一次截屏结果保存于剪切板，右键粘贴即可保存；
    5.按ESC键可退出截图并关闭当前截图窗口。

已知bug:
    按 Ctrl + Z 后直接按 ESC 导致程序不可用
